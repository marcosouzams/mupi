12:Tf72,<p>Prezados clientes e usuários,</p>
<p>Nesse mês de julho gostaríamos de destacar nosso projeto de monitoramento de redes sociais. Estamos acompanhando pelo Twitter as principais métricas de alguns potenciais candidatos para as eleições 2022.  Veja a postagem da nossa primeira análise na nossa conta do <a href="https://twitter.com/TextualizaDados/status/1423040107406319616" target="_blank" rel="noopener">twitter</a>  (@textualizadados). Também contamos com a colaboração de vocês com sugestão de novas contas ou temas para análise!</p>
<p>Sobre a plataforma <a href="https://minhaagendavirtual.com.br/" target="_blank" rel="noopener">Minha Agenda Virtual,</a> começamos a divulgar a integração usando nossa API pública. Com isso, desenvolvedores de sistemas conseguirão criar agendamentos ou links de agendamento integrados com outros sistemas. Exemplos de aplicações são a criação de ferramentas de agendamento via chatboot, gerar link de agendamento para um cliente que fez uma compra ou gerar link de agendamentos para clientes que possuem algum plano de agendamento (ex: alunos de academias, pacientes de fisioterapia, terapia, etc). Todas essas funcionalidades estão disponíveis para uso por meio das telas da plataforma, porém a integração via API permite automatizar todas essas tarefas manuais.</p>
<p>Além disso, trabalhamos na construção de fluxos de navegação para permitir que os usuários encontrem a agenda correta de um modo mais simples. Por exemplo, é possível construir etapas com escolhas simples e ir navegando até chegar ao final. Por exemplo, se você tem 20 agendas, pode criar uma primeira etapa com 2 escolhas (grupo A ou B). Em seguida, uma nova etapa com mais duas opções. Por exemplo, se o cliente escolheu o grupo A na primeira etapa, agora pode escolher entre o grupo A.1 e A.2. Selecionando esse grupo final, a escolha fica reduzida para duas ou três agendas. Sem esse fluxo para agrupar, o cliente pode ter que escolher já no passo inicial entre as 20 agendas e isso é muito ruim do ponto de vista da usabilidade do sistema. Essa nova funcionalidade nova está disponível para todos os planos da plataforma.</p>
<p>Na plataforma <a href="https://minhasalavirtual.com/" target="_blank" rel="noopener">Minha Sala Virtual</a>, trabalhamos em melhorias no processo de gamificação dos cursos e treinamentos. Vamos fazer uma publicação específica sobre como as técnicas de gamificação podem ser úteis para melhorar a produtividade e engajamento da equipe no processo de aprendizagem. Por exemplo, é possível criar um curso com uma sequencia de aulas semanais e vincular algumas tarefas que os participantes devem concluir antes da próxima aula. A aula seguinte só é liberada quando o participante conclui as tarefas e há uma pontuação conforme as tarefas vão sendo concluídas.</p>
<p><span style="font-size: revert; color: var(--nv-text-color); font-family: Graphik,-apple-system,BlinkMacSystemFont,Segoe UI,var(--nv-fallback-ff); background-color: var(--nv-site-bg);">Na plataforma </span><a style="font-size: revert; font-family: Graphik,-apple-system,BlinkMacSystemFont,Segoe UI,var(--nv-fallback-ff); background-color: var(--nv-site-bg);" href="https://etalentos.com.br/" target="_blank" rel="noopener">E-Talentos</a><span style="font-size: revert; color: var(--nv-text-color); font-family: Graphik,-apple-system,BlinkMacSystemFont,Segoe UI,var(--nv-fallback-ff); background-color: var(--nv-site-bg);">, estamos trabalhando em melhorias na usabilidade geral da plataforma e novas funcionalidades como a criação de formulários de inscrição personalizados para cada processo. Ou seja, se para um processo seletivo houver um requisito específico, o recrutador pode criar um formulário de inscrição customizado e analisar se os candidatos atendem àqueles requisitos.</span></p>
<p>Por hoje é isso! Desejamos um excelente mês a todos!</p>
